1. รูปแบบของ Lambda Function

Lambda function หรือที่เรียกว่า anonymous function (ฟังก์ชันไร้ชื่อ) คือฟังก์ชันที่ถูกสร้างขึ้นโดยใช้คำสั่ง lambda ใน Python โดยที่ฟังก์ชันนี้จะไม่มีชื่อ และมักจะใช้ในกรณีที่ฟังก์ชันนั้น ๆ ไม่ได้มีความซับซ้อนหรือใช้แค่ครั้งเดียว

lambda arguments: expression

ตัวอย่างเช่น:

add = lambda x, y: x + y
print(add(2, 3))  # ผลลัพธ์คือ 5

2. การใช้งาน Lambda Function

Lambda function มักจะใช้ในสถานการณ์ที่ต้องการฟังก์ชันง่าย ๆ สำหรับการประมวลผลบางอย่าง โดยเฉพาะเมื่อใช้ร่วมกับฟังก์ชันใน Python เช่น map(), filter(), sorted() ฯลฯ

ตัวอย่างการใช้ Lambda กับ map():

numbers = [1, 2, 3, 4]
squared = list(map(lambda x: x**2, numbers))
print(squared)  # ผลลัพธ์: [1, 4, 9, 16]

3. ข้อดีและข้อเสียของ Lambda Function

ข้อดี: Lambda function ช่วยให้โค้ดของคุณกระชับและสั้นลง เหมาะสำหรับใช้ในกรณีที่ต้องการฟังก์ชันชั่วคราวและไม่ซับซ้อน

ข้อเสีย: การใช้ Lambda อาจทำให้โค้ดอ่านยากขึ้น โดยเฉพาะเมื่อมีความซับซ้อนมากขึ้นหรือใช้ในหลาย ๆ จุด

4. ตัวอย่างการใช้งานเพิ่มเติม

ตัวอย่างการใช้ Lambda กับฟังก์ชันต่าง ๆ ใน Python:

  • filter(): ใช้เพื่อกรองข้อมูลจากลิสต์
    • numbers = [1, 2, 3, 4, 5, 6]
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # ผลลัพธ์: [2, 4, 6]
  • sorted(): ใช้ในการจัดเรียงข้อมูล
    • numbers = [5, 2, 9, 1, 5, 6]
sorted_numbers = sorted(numbers, key=lambda x: x)
print(sorted_numbers)  # ผลลัพธ์: [1, 2, 5, 5, 6, 9]
